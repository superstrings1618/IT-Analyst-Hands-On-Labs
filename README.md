# IT Analyst Hands-On Labs

This repository documents hands-on practice aligned with an Information Technology Analyst (Trainee) role.

## Skills Demonstrated
- Windows Administration
- Active Directory User & Group Management
- IT Troubleshooting Methodology
- Networking Fundamentals
- Ticketing Systems & Documentation
- Security Best Practices

## Environment
- Windows 10 / Windows Server
- VirtualBox
- Kali Linux
- Active Directory Domain Services

## Purpose
All labs are documented with scenarios, steps taken, tools used, and verification to simulate real-world IT support environments.
